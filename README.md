# Resume

A fully customisable online resume created with [Next.js](https://nextjs.org/).

## Features

* Customisable
* Semantic HTML
* [h-resume](http://microformats.org/wiki/h-resume) microformat
* Search Engine Optimised
* Perfect Lighthouse score
